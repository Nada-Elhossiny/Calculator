# Calculator
A basic GUI calculator using Tkinter in `Python`.

The Calculator supports 
- Addition
- Subtraction
- Multiplication
- Division
- Percentages
- Exponents 
- Decimals
- Clearing Screen
- Ability to detect errors (e.g. divide by zero)
